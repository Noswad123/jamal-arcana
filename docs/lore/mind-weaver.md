# Mind Weaver

![Mind Weaver](../../img/mind-weaver.png)

The MindWeaver is a cosmic being that spins thoughts into threads using his [Astral Loom](./celestial-loom.md). It does not create knowledge directly; it binds fragments together so they can be found again.


## Classification

- Jamal-Forged Arcana
- Cosmic Being

## Mundane Form

- A notes, todos, cheatsheets, and local knowledge system.

## Arcane Form

- A living loom of memory that binds fragments into retrievable threads.

## Domains

- Knowledge
- Memory
- Thought
- Notes
- Tasks
- Recall

## Relationships

- Uses the [Celestial Loom](./celestial-loom.md), also known in early notes as the Astral Loom.
- Connects naturally with [Ideomancer](./ideomancer.md), which structures ideas before or after they are woven.
