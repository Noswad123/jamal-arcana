# Hexware

## Classification

- Native
- Design substrate
- Interface magic

## Veiled Form

- A shared visual theme language for Jamal Arcana tools: reusable color tokens, surfaces, accents, borders, and theme variants that can travel across web, SwiftUI, terminal, and other GUI layers.

## Arcane Form

- Hexware is the magical-technical substrate used to clothe tools in visible power.
- Where Hexware is the magical-technical substrate, the first word is the enchantment, mood, domain, or elemental attribute imbued into it.
- The name carries a double meaning: **hex** as spellcraft and **hex** as hexadecimal color, joining fantasy naming with practical interface tokens.

## Named Imbuements

| Imbuement | Mood | Interface role |
| --- | --- | --- |
| Astral Hexware | Cosmic, luminous, mysterious | Default Jamal Arcana dark theme |
| Ember Hexware | Warm, decisive, forge-lit | Alerts, action-heavy tools, creation flows |
| Frost Hexware | Cool, crystalline, analytical | Navigation, systems, dashboards, focused analysis |
| Verdant Hexware | Living, restorative, knowledge-growing | Notes, recipes, learning, gardens, renewal |

## Design Tenets

- Favor dark cosmic foundations with readable foregrounds.
- Use luminous accents as signals, not decoration everywhere.
- Treat surfaces like enchanted panels: soft edges, subtle borders, and calm hierarchy.
- Keep the theme portable by naming semantic tokens before implementation-specific colors.
- Reserve **Sigiltech** for future typography, glyphs, icons, or symbolic interface systems.

## Relationships

- **Mimic** previews and manages Hexware imbuements before they are carried into other tools.
- **Mind Weaver** is a natural host for Astral or Verdant Hexware because it works with knowledge, notes, and living thought structures.
- **Sigiltech** may eventually become the glyphic layer that sits on top of Hexware.
